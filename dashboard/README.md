# Power BI Dashboard Pack

This folder contains the implementation blueprint for the RetailIQ executive dashboard.

## Files
- `PowerBI_Dashboard_Spec.md`: page-by-page dashboard specification
- `PowerBI_Wireframe.md`: ASCII wireframes for each page
- `PowerBI_Measures.md`: suggested DAX measures for the executive layer

## Intended Use
Open these files while building the dashboard in Power BI Desktop. They define:
- visual selection
- page structure
- KPI cards
- slicers and filters
- navigation
- color system
- measure set

## Leadership Style
The intended audience is executive leadership at a large e-commerce marketplace. The layout should feel premium, concise, and operationally useful.
